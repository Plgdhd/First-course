color: Определяет цвет текста элемента. Например, можно установить цвет текста для абзаца в синий.
background-color: Задает цвет фона элемента. Это свойство позволяет изменить фон, например, на светло-серый для блока div.
font-size: Устанавливает размер шрифта для текста. Это свойство помогает задать размер заголовка, например, 24 пикселя для h1.
margin: Определяет внешние отступы элемента. Можно установить одинаковые отступы со всех сторон, например, 10 пикселей для класса box.
padding: Устанавливает внутренние отступы элемента, то есть пространство между содержимым и границей. Например, можно задать 20 пикселей внутренних отступов для контейнера.
border: Определяет границу элемента, включая ширину, стиль и цвет. Например, можно задать черную границу толщиной 2 пикселя для изображений.
display: Определяет способ отображения элемента, например, блочный, строчный или гибкий. Используя это свойство, можно задать, что контейнер будет использовать flexbox для расположения дочерних элементов.
position: Определяет способ позиционирования элемента, например, static, relative, absolute, fixed или sticky. Можно задать абсолютное позиционирование для элемента с заданными координатами.
width и height: Устанавливают ширину и высоту элемента. Например, можно задать ширину 200 пикселей и высоту 100 пикселей для элемента класса box.
overflow: Определяет, что будет происходить с содержимым элемента, если оно превышает его размеры. Это свойство позволяет добавить прокрутку, если содержимое выходит за границы элемента.
